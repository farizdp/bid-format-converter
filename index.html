BID Format Converter

Input

Enter numbers or text (one per line). Numbers will be padded to 8 digits, and text will be converted to lowercase.

Output

Formatted result with proper padding and quotes

Examples

Number Input Example:

Input:
123456
234567
1

Output:
'00123456',
'00234567',
'00000001'

Text Input Example:

Input:
Netflix 89000
Amazon Prime VIDEO

Output:
'netflix 89000',
'amazon prime video'
